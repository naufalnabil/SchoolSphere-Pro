# SchoolSphere-Pro

Website sekolah modular berbasis **PHP Native Modular + Laragon + MySQL/phpMyAdmin** dengan pendekatan **frontend-first**.

Mulai dari dokumen:

```txt
docs/00_README_START_HERE.md
```

Folder ini disiapkan untuk diletakkan di:

```txt
C:\laragon\www\SchoolSphere-Pro
```

Buka folder ini di Antigravity, lalu minta agent membaca folder `docs/` sebelum mengubah kode.
